Tic-Tac-Toe AI

Objective

Develop an AI agent that plays Tic-Tac-Toe against a human player using the Minimax algorithm.

Features

- Human vs AI gameplay
- AI chooses the best possible move
- Win, lose, and draw detection
- Console-based interface

Technologies Used

- Python

Algorithm

- Minimax Algorithm

Outcome

The AI evaluates all possible moves and selects the optimal move, making it unbeatable.
